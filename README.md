# Stream Gauge Dataset v2

100 high-quality stream gauge / staff gauge images for UNet binary classification training.

## Dataset
- **Images:** 100
- **Avg Resolution:** ~1110×1660 px
- **Sources:** Compiled from Roboflow Universe datasets
- **Purpose:** Train a UNet model for binary segmentation of water level staff gauges
